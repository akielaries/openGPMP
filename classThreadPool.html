Detailed Description

A thread pool implementation for concurrent task execution. This class provides a thread pool implementation that can execute tasks in parallel on multiple threads. It supports adding tasks to the pool and retrieving their results using futures. The number of threads in the pool can be specified or it can be automatically determined using std::thread::hardware_concurrency(), therefore using maximum amount of threads on the system. The pool is stopped and all threads are joined on destruction of the object.
